# INTERNSHIP---Task-1-Excel Data Cleaning
Dataset: netflix_titles-selected-columns.csv
Records: 8,807 rows  ×  12 columns

Cleaning Steps Performed :
1. Missing Value Treatment  -   Identified blank cells across the dataset using
Go To Special → Blanks, and filled them with the text value “null” 
using Ctrl + Enter -  to apply it uniformly to all selected blank cells in one action

2. Text Standardization -  Standardized inconsistent text entries across categorical/text columns (case inconsistencies, varied formatting) to bring uniformity to the dataset's text values.

Tools & Techniques Used :
•	Go To Special (F5 → Special → Blanks) to isolate empty cells
•	Ctrl + Enter to bulk-fill all selected blank cells in a single step
•	Text case and formatting standardization across relevant columns

Outcome
After these two steps, the dataset no longer has empty cells : all blanks are now marked as “null”
Text fields follow a consistent format : improving readability and making the data more reliable for further analysis.
